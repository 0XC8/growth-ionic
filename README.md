##简介

Growth来源于: 对Web应用开发的一个流程认识。

主要关注点是**Web开发的流程及其技术栈、学习路线、成长衡量**，包含：

 - 完整的Web开发流程介绍   
 - 如何写好代码——重构、测试、模式
 - Web应用的分析和部署
 - 遗留代码、遗留系统的形成
 - 每一阶段所需的技能

##FAQ

**为什么没有WP和iOS版?**

- 没有iPhone和iOS开发者帐号
- 没有Windows机器的电脑(平时用的是公司的电脑)

**我可以复制吗?**

> 想法和代码是可以复制的，但是内容是不行的。
